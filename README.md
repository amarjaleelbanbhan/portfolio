## ğŸš€ Quick Start

### Prerequisites

- Node.js 18+ installed
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/amarjaleelbanbhan/portfolio.git

# Navigate to project

# Install dependencies
```
